隐私权政策

最后更新：2026 年 3 月

核心承诺

Lockin 是一款纯本地运行的阅读辅助扩展。我们不收集、不传输、不存储您的任何个人数据。您的阅读内容和浏览行为完全属于您自己。

1我们收集哪些数据

无。 Lockin 不收集任何个人身份信息、浏览历史、页面内容或用户行为数据。

2本地存储的使用

Lockin 使用 Chrome 的 chrome.storage.sync API 来保存您的偏好设置（如功能开关状态、滑块数值、主题选择、语言偏好等）。

这些数据仅用于在您的 Chrome 设备之间同步设置，完全由 Chrome 浏览器自身管理，Lockin 无法也不会将这些数据发送至任何外部服务器。

3网络请求

Lockin 仅在启用「字体切换」功能时，可能从 Google Fonts 或 jsDelivr CDN 加载字体文件。除此之外，Lockin 不会发起任何网络请求。

4权限说明

activeTab — 仅在用户主动激活时访问当前标签页，用于在页面上应用阅读辅助效果（如 Bionic Reading、行高亮等）。

storage — 用于保存和同步用户的偏好设置。

5第三方服务

Lockin 不使用任何分析工具（如 Google Analytics）、广告服务或第三方追踪器。

6数据安全

由于所有数据都存储在您本地的浏览器中，不存在数据泄露的风险。Lockin 不运行任何远程服务器。

7儿童隐私

Lockin 不针对 13 岁以下的儿童，也不会故意收集任何儿童的个人信息。

8政策变更

如果本隐私政策发生重大变更，我们会在此页面更新并修改「最后更新」日期。建议您定期查看。

Privacy Policy

Last updated: July 2025

Our Promise

Lockin is a fully local reading-assistance extension. We do not collect, transmit, or store any of your personal data. Your reading content and browsing behavior remain entirely your own.

1Data We Collect

None. Lockin does not collect any personally identifiable information, browsing history, page content, or user behavior data.

2Local Storage Usage

Lockin uses Chrome's chrome.storage.sync API to save your preferences (feature toggles, slider values, theme, language, etc.).

This data is only used to sync settings across your Chrome devices and is managed entirely by the Chrome browser itself. Lockin cannot and does not send this data to any external server.

3Network Requests

When the "Font Switcher" feature is enabled, Lockin may load font files from Google Fonts or jsDelivr CDN. Apart from this, Lockin does not make any network requests.

4Permissions Explained

activeTab — Accesses the current tab only when the user actively enables the extension, in order to apply reading aids (Bionic Reading, line highlight, etc.).

storage — Used to save and sync user preferences.

5Third-Party Services

Lockin does not use any analytics tools (such as Google Analytics), advertising services, or third-party trackers.

6Data Security

Since all data is stored locally in your browser, there is no risk of data breach. Lockin does not operate any remote servers.

7Children's Privacy

Lockin is not directed at children under 13 and does not intentionally collect any personal information from children.

8Changes to This Policy

If we make significant changes to this privacy policy, we will update this page and revise the "Last Updated" date. We recommend checking this page periodically.
